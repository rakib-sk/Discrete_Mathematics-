### Definition  
A **Compound Proposition** is a proposition formed by combining two or more simple propositions using logical operators (connectives).

### Example
1. p = It is raining
2. q = I will stay at home
3. Compound proposition:
4. p AND q ‚Üí It is raining AND I will stay at home
